# javascript-apps

repo for random java script apps

## Create new application folders

`mkdir new-app && cd new-app`

## init new project

`npm init -y`

## Clean install a project

npm ci

## Install a package

npm install

## Compile

tsc or tsc-w  `to watch`

## Run JS

node build/index.js

## install NodeMon & Concurrently

npm install nodemon concurrently --save-dev

## start process

npm start
