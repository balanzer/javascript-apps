# MAPS App

## Clean install a project

npm ci

## Install a package

npm install

## Install Parcel

npm install --save-dev parcel

## Run with Parcel

npx parcel src/index.html

and try http://localhost:1234/

## Install Faker

npm install @faker-js/faker --save-dev

## install google map type

npm i -D @types/google.maps
